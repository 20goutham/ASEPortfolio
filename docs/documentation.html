1. Project Overview

This is a personal developer portfolio built to showcase professional experience, projects, skills, and provide contact channels for recruiters and collaborators. It is designed to be visually impressive, performant, and fully SEO-optimised.

Live URL: https://khatravatgoutham.netlify.app
GitHub Repo: https://github.com/20goutham/ASEPortfolio
Deployment: Netlify (auto-deploys on every push to main)

3. Features

🌌 Cosmos Preloader
3.5-second welcome screen with starfield, nebula clouds, shooting stars, and animated KGR logo.
🎨 8-Theme Auto-Cycle
8 blue-variant themes that rotate every 7 seconds. Manual cycle by clicking the theme button in the navbar.
🌐 TechSphere 3D Globe
Canvas-based rotating 3D sphere behind the profile photo, built with Fibonacci lattice and perspective projection.
🎙️ Voice Introduction
Web Speech API voiceover using a male Indian English voice. Supports play, pause, and resume.
🧲 Magnetic Buttons
Hero CTA buttons magnetically attract to the cursor via RAF-based lerp animation for a premium feel.
🖱️ Custom Cursor Trail
8px dot + 26px ring + 8 trailing dots, all RAF-driven with cascade lerp. Enlarges on hover over interactive elements.
📬 Contact + Feedback Forms
Contact form sends to Gmail via EmailJS. Exit-feedback form captures recruiter input on tab close.
📅 Schedule a Call
Modal with Calendly booking link, WhatsApp, and phone options for easy scheduling.
👁️ Visitor Counter
Real-time hit count via counterapi.dev. Starts from 15 via offset. Shown in the Footer.
⚡ Lazy Loading
All below-fold sections use React.lazy() + Suspense. 16 separate JS chunks for fast initial load.
🔍 Full SEO
Canonical URL, Open Graph, Twitter Card, JSON-LD Person schema for Google Knowledge Graph.
🟢 Availability Badge
Pulsing green ping dot + border glow animation on "Available for Opportunities" in the navbar.

5. Local Development Setup

1
Clone the repository 
git clone https://github.com/20goutham/ASEPortfolio.git
cd ASEPortfolio
2
Install dependencies 
npm install
3
Create environment file

Copy .env.example to .env and fill in your EmailJS credentials:

VITE_EMAILJS_SERVICE_ID=service_xxxxxxx
VITE_EMAILJS_TEMPLATE_ID=template_xxxxxxx
VITE_EMAILJS_PUBLIC_KEY=xxxxxxxxxxxxxxxxxxxx
4
Start development server 
npm run dev

Opens at http://localhost:5173

5
Build for production 
npm run build

Output goes to dist/ folder.

6. Environment Variables

⚠️ The .env file is listed in .gitignore and is never committed to GitHub. Set these variables in Netlify → Site Settings → Environment Variables.

7. Updating Content

All personal data is centralised in src/constants/index.js. Edit this file to update:

  • Name, role, location, email, phone
  • LinkedIn, GitHub, WhatsApp links
  • Typewriter taglines
  • Navigation links

Adding a New Project

Open src/components/Projects.jsx and add an entry to the PROJECTS array:

{
  title:  'Project Name',
  desc:   'Short description of what it does.',
  tech:   ['Python', 'React', 'AWS'],
  github: 'https://github.com/...',
  live:   'https://...',
}

Adding a New Experience

Open src/components/Experience.jsx and add to the EXPERIENCES array:

{
  role:     'Job Title',
  company:  'Company Name',
  period:   'Jan 2024 – Present',
  points:   ['Achievement 1', 'Achievement 2'],
}

Updating Skills

Open src/components/Skills.jsx — update the skills array with name and level (0–100).

Updating Resume Files

Replace the HTML files in public/resumes/:

  • resume.html — General resume
  • resume-fullstack.html — Full Stack variant
  • resume-aiml.html — AI/ML variant
  • resume-dataengineer.html — Data Engineering variant

10. Deployment Guide

Automatic Deployment (current setup)

Every push to the main branch on GitHub automatically triggers a Netlify rebuild. No manual steps needed.

1
Make changes locally

Edit files in src/ or public/

2
Commit in GitHub Desktop

Write a commit message → click Commit to main

3
Push to GitHub

Click Push origin in GitHub Desktop

4
Netlify auto-deploys

Build completes in ~1–2 minutes. Live at khatravatgoutham.netlify.app

Netlify Configuration (netlify.toml)

[build]
  command = "npm run build"
  publish = "dist"

[[redirects]]
  from   = "/*"
  to     = "/index.html"
  status = 200         # SPA fallback — all routes serve index.html

[build.environment]
  NODE_VERSION = "18"

11. SEO Configuration

All SEO tags are in index.html. Key elements:

  • Canonical URL — https://khatravatgoutham.netlify.app
  • Open Graph — Controls how the link looks when shared on LinkedIn, WhatsApp, Facebook
  • Twitter Card — Controls the preview on X / Twitter
  • JSON-LD Person Schema — Helps Google show a Knowledge Panel with your name, role, and links
If you get a custom domain later — replace all occurrences of khatravatgoutham.netlify.app in index.html with your new domain, then push.
